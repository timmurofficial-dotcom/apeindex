## Slash Commands

| Command | Description |
|---------|-------------|
| `/start` | Open the main menu |
| `/monitor` | Start or stop the monitoring session |
| `/portfolio` | View all open positions with live P&L |
| `/watchlist` | View and manage your watched markets |
| `/markets` | Open the market browser |
| `/wallets` | Manage followed whale wallets |
| `/settings` | Configure risk, patterns, trading mode, and API keys |
| `/digest` | Request an on-demand daily digest (any time) |
| `/help` | Show command help |

## Main Menu Buttons

The main menu (shown after `/start`) provides one-tap access to the most common actions:

<CardGroup cols={2}>
  <Card title="▶ Start Monitoring" icon="play">
    Activates the background monitoring task. Signals begin arriving within the next 30-second cycle.
  </Card>
  <Card title="⏹ Stop Monitoring" icon="stop">
    Stops the monitoring task. No new signals will fire until monitoring is restarted.
  </Card>
  <Card title="📂 Portfolio" icon="folder">
    Shortcut to `/portfolio` — real-time open positions view.
  </Card>
  <Card title="⚙️ Settings" icon="gear">
    Shortcut to `/settings` — risk, patterns, API keys, timezone.
  </Card>
  <Card title="🐳 Wallets" icon="fish">
    Shortcut to `/wallets` — add or remove tracked whale addresses.
  </Card>
  <Card title="📊 Markets" icon="chart-line">
    Shortcut to `/markets` — trending, category browse, keyword search.
  </Card>
</CardGroup>

## Signal Notification Buttons

Every signal notification includes inline buttons:

| Button | Action |
|--------|--------|
| **Enter Trade** | Opens trade entry with pre-filled market, outcome, and suggested size |
| **Skip** | Dismiss the signal without action |
| **Add to Watchlist** | Track the market's price movement without trading |
| **View Market** | Open the market detail screen |

## Trade Confirmation Buttons

After entering a trade (Manual Mode), you see:

| Button | Action |
|--------|--------|
| **Confirm** | Execute the trade via CLOB API |
| **Cancel** | Abort without executing |
| **Change Size** | Adjust the trade amount before confirming |
