## Connecting Your Wallet

No external setup required. ApeIndex generates a fully functional trading wallet for you directly inside Telegram — including private key, CLOB API credentials, and a deposit address.

<Steps>
  <Step title="Create Your Wallet">
    In the bot, tap Wallet → Create Wallet. The terminal instantly generates a new Polygon wallet with all required keys. No browser extension or external app needed.
  </Step>
  <Step title="Deposit USDC">
    The terminal shows your deposit address. Send USDC (Polygon network) to that address. Your balance will appear in the bot automatically once the transaction confirms.
  </Step>
  <Step title="Start Trading">
    That's it — your wallet is ready. All CLOB API credentials are generated and stored automatically. You can start receiving and executing signals immediately.
  </Step>
</Steps>

<Warning>
  Save your private key shown during wallet creation — it is displayed only once. ApeIndex stores it encrypted, but you should back it up in a safe place in case you need to recover access.
</Warning>

## Risk Configuration

Risk settings apply at two levels:

### Global Settings

Applied to all patterns unless overridden by per-pattern settings.

| Setting              | Default | Description                         |
| -------------------- | ------- | ----------------------------------- |
| Minimum trade amount | \$10    | Smallest amount executed per signal |
| Maximum trade amount | \$100   | Cap on any single trade             |
| Risk level           | Medium  | Multiplier applied to base sizing   |

### Per-Pattern Settings

Each of the 5 patterns can have independent sizing. Configure via `/settings → Per-Pattern Risk`.

### Risk Levels

<CardGroup cols={3}>
  <Card icon="shield" title="Low">
    Conservative sizing. Best for new users or volatile markets.
  </Card>
  <Card icon="arrows-minimize" title="Medium">
    Balanced. The recommended starting point. 
  </Card>
  <Card icon="fire" title="High">
    Maximum sizing. Only use with patterns you have confidence in.
  </Card>
</CardGroup>

## Notification Settings

Control which patterns send notifications and whether they are auto-executed:

```text
/settings → Signal Patterns
```

Toggle each pattern on/off independently for both **notifications** and **auto-execution**.
